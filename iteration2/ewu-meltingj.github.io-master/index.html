My name is Jeremy Meltingtallow

I am a 27yr old human being with a passion for design. I have been active in the web development scene for over 14 years.

Design Philosophy

Focus on a clean interface where the user needs not to think. Color has its place but one should defer from a loud design if it takes away from a transparent user experience. Basically I design as if the user is drunk.

Beyond editing, my other favorite tools include Nodejs, npm, Angularjs, Cordova, and jekyll. This excludes countless other tools.
